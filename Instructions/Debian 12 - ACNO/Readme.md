Apache coturn Nextcloud Office / Acronym "ACNO" or just "ao".

Setup and maintain a server, which is going to become accessible from the internet. Hosted at home.

These instructions are for Debian 12 running the components as they are delivered and trying to keep things as simple as possible. However some security aspects ave to be implemented, as the system is going to be "exposed" to the internet.

I do have created these instruction 

1. on my endeavor to setup a Nextcloud system that fits my needs,
2. keep it running with minimum effort and
3. be able to restore/rebuild it at anytime.

People should avoid these instructions who

- look for a quick and easy solution
- want something based on hypervisors or container
- have a strong opinion on what is the best, which is in contrast to the used components/software and structure
